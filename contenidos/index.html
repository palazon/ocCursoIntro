OpenCode 101 — Curso básico para creadores de contenido

¿Por qué OpenCode para documentación?

OpenCode es un asistente de inteligencia artificial que trabaja directamente sobre tus archivos locales. Aunque nació como herramienta para desarrolladores, es igualmente útil para cualquier persona que trabaje con textos y documentos.

¿Qué hace OpenCode por ti?

  • Lee y escribe documentos Word, Markdown, PDFs y texto plano, pero para trabaja usa estos últimos
  • Mejora la redacción, el tono y la claridad de tus textos
  • Traduce documentos manteniendo el formato original
  • Busca información actualizada en internet
  • Reestructura, resume y transforma contenido sin que tengas que copiar y pegar manualmente

A diferencia de ChatGPT o Gemini, OpenCode trabaja en tu propio ordenador, sobre tus propios archivos. No subes información a una web: el asistente abre, lee, modifica y guarda los documentos directamente en tu disco.

Cómo usar este curso

Estás en el índice del curso. Desde aquí puedes seguir los enlaces a cada paso.

  1. Sigue el orden — del 0 al 11. Cada paso se apoya en el anterior.
  2. Ten OpenCode abierto en la carpeta del proyecto mientras lees.
  3. Acumula archivos — los documentos se reutilizan entre pasos.
  4. Plan para pensar, Build para hacer — explora sin riesgo en Plan, ejecuta cambios en Build.
AvisoEl proyecto es una carpeta (o directorio)

Todo el curso se organiza en torno a proyectos. Un proyecto no es más que una carpeta (también llamada directorio) en tu ordenador donde irás creando y guardando todos los archivos. Piensa en ella como tu espacio de trabajo: OpenCode guardará ahí tus documentos, conversaciones y configuración.

ConsejoPara usuarios avanzados

Los archivos .qmd de la carpeta contenidos/ son el formato fuente del curso. Puedes abrirlos con OpenCode, modificarlos y regenerar los HTML con Quarto si quieres personalizar el contenido.

Público y requisitos

¿Para quién es este curso?

  • Docentes que preparan materiales didácticos
  • Redactores técnicos que escriben manuales, guías o informes
  • Personal de administración que genera documentación
  • Cualquier persona que trabaje con textos de forma habitual

No se necesita saber programar. Todo se hace mediante instrucciones en lenguaje natural.

Requisitos técnicos

  • OpenCode instalado (se explica en el Paso 2)
  • Una cuenta gratuita en OpenCode Zen (viene activada por defecto)
  • Un navegador web para ver los HTML generados
  • Quarto (opcional, para regenerar los HTML si se desea)
